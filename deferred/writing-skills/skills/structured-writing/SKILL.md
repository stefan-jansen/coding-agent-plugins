# Structured Writing Frameworks

Three proven frameworks for organizing written content effectively.

## 1. Pyramid Principle (Barbara Minto)

**Core**: Answer first, then supporting arguments, then details.

```
        [ANSWER]
           |
   [Arg 1] [Arg 2] [Arg 3]
      |       |       |
  [Details] [Details] [Details]
```

**Rules**:
- Start with conclusion (reader gets point immediately)
- Group related ideas at same abstraction level
- Use logical order (structural, chronological, comparative, problem-solution)

## 2. SCQA Framework (McKinsey)

**Core**: Build narrative tension that primes reader for your answer.

| Element | Purpose | Example |
|---------|---------|---------|
| **S**ituation | Common ground | "Teams use AI assistants daily" |
| **C**omplication | Problem/tension | "But generic AI produces unreliable code" |
| **Q**uestion | What reader asks | "How to get reliability?" |
| **A**nswer | Your solution | "Domain-specific customization" |

**Use for**: Opening hooks, persuasive content, problem-solution framing.

## 3. Topic Sentence Method

**Core**: Each paragraph's first sentence states its point and connects to hierarchy.

**Five patterns**:
1. **Definition + Importance**: "[Concept] is [definition]. This [impact]..."
2. **Action + Result**: "[Do X] to [achieve Y]. This [outcome]..."
3. **Problem + Solution**: "[Problem] occurs when... [Solution] resolves..."
4. **Concept + Analogy**: "[Abstract] works like [familiar]..."
5. **Question + Answer**: "[Question]? [Answer]. [Evidence]..."

**Quality criteria**: Explicit claims, connected to hierarchy, standalone (scannable), evidence-ready.

## When to Use

| Framework | Best For |
|-----------|----------|
| Pyramid | Outlines, business communication, technical explanations |
| SCQA | Opening paragraphs, persuasive narratives, problem framing |
| Topic Sentence | Paragraph-level drafting, expanding outlines to prose |

## Integration

```
SCQA (opening) → Reader primed for answer
    ↓
Pyramid (body) → Answer + supporting arguments hierarchically
    ↓
Topic Sentences (paragraphs) → Each paragraph advances hierarchy
```
